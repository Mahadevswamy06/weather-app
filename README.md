# Weather App

> A premium, modern, and production-ready weather application built with React.js using the Open-Meteo API.

## üöÄ Features

*   **Search Functionality**: Search for any city worldwide to get instant weather updates.
*   **Current Weather**: Detailed view including Temperature, Humidity, Wind Speed, and Atmospheric Pressure.
*   **5-Day Forecast**: Accurate daily forecast to help you plan your week.
*   **Geolocation**: Automatically detects your location on startup to show local weather.
*   **Dynamic Backgrounds**: The application interface changes color and theme based on the current weather conditions (Sunny, Rainy, Cloudy, etc.).
*   **Glassmorphism Design**: Modern, sleek UI with translucent cards and smooth animations.
